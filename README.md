# Angular Quickstart

## Installation

    $ cd [PROJECT_DIR]
    $ npm run typings install
    $ npm install
    
## Running In Development Mode

    $ cd [PROJECT_DIR]
    $ npm run develop
    
## Running In Production Mode

    $ npm run build
    $ npm start
